# 加入服务器
## 在开始前
各位玩家们好，小服务器是很难生存的，不仅会面临病毒入侵、外挂黑服，还会遭遇 DDoS 攻击，甚至是 UDP 洪水攻击。  
恳请各位技术人员手下留情！

## 本服概览
!!! note "本服配置"

    CPU: Intel(R) Core(TM) i9-14900KF CPU @ 3.2GHz  
    RAM: 8.0 GB    

目前服务器使用 [Fabric](https://fabricmc.net) 驱动。Java 和基岩版支持的游戏版本分别为  `1.21.11`, `1.21.111 - 26.0`.

* 服务器每天备份以防有人破坏服务器秩序。

## 本服宗旨
主打慢节奏**抱团**原版生存，如果要做一只沉默的独狼，  
那进入服务器有什么意义呢？

## 规则
我们禁止以下行为：
***

- 修改游戏数据（使用辅助程序等）
- 使用不合理的资源包（矿物透视等）
- 制造会使服务器资源过载的建筑
- 损害他人利益

\* 损害他人利益：包括但不限于窃取财产、破坏建筑和攻击性言论等，具体范围由我们根据情节决定。
